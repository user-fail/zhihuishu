## 使用

**每门课回答40个问题**

登录
1. 用扫码登录直接运行
2. 用账户密码登录需要配置
```angular2html
{
  "isPasswordLogin":true, 使用账户密码登录,默认为false
  "account": "",账号
  "password": ""密码
}
```
直接运行main.py
